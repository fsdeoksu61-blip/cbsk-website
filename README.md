# 한국교회 상담센터 웹사이트 리뉴얼

## 📋 개요

기존의 테이블 기반 레이아웃에서 현대적인 반응형 웹사이트로 전면 리뉴얼했습니다.

## ✨ 주요 개선사항

### 1. 현대적인 디자인
- **프로페셔널한 느낌**: 신뢰감 있는 색상과 레이아웃
- **깔끔한 UI**: 여백과 타이포그래피를 활용한 가독성 향상
- **일관된 디자인**: 통일된 디자인 시스템 적용

### 2. 반응형 웹 디자인
- **모바일 최적화**: 스마트폰, 태블릿, PC 모두 지원
- **반응형 메뉴**: 모바일에서는 햄버거 메뉴로 변환
- **유연한 레이아웃**: 화면 크기에 따라 자동으로 조정

### 3. 최신 웹 기술 적용
- **HTML5**: 시맨틱 태그 사용으로 접근성 향상
- **CSS3**: Flexbox, Grid, 애니메이션 활용
- **Modern JavaScript**: ES6+ 문법, 클래스 기반 구조

### 4. 성능 및 접근성
- **빠른 로딩**: 최적화된 코드 구조
- **검색엔진 최적화(SEO)**: 적절한 메타 태그와 시맨틱 마크업
- **접근성**: ARIA 레이블, 키보드 네비게이션 지원

## 🔧 주요 기능

### 1. 반응형 네비게이션
- PC: 가로 메뉴
- 모바일: 햄버거 메뉴
- 스크롤 시 고정 헤더

### 2. 히어로 슬라이더
- 5초마다 자동 슬라이드
- 부드러운 전환 효과
- 오버레이 텍스트

### 3. 퀵 링크 섹션
- 4개의 주요 링크 카드
- 호버 효과 (카드 상승)
- 아이콘과 설명 포함

### 4. 콘텐츠 섹션
- 공지사항과 칼럼 목록
- 깔끔한 리스트 디자인
- 날짜 정보 표시

### 5. 서비스 섹션
- 3개의 주요 서비스 영역
- 이미지와 텍스트 조합
- 성경 구절 인용

### 6. 부가 기능
- **맨 위로 버튼**: 스크롤 시 자동 표시
- **스크롤 애니메이션**: 요소가 나타날 때 페이드 인
- **스무스 스크롤**: 앵커 링크 부드러운 이동

## 🚀 사용 방법

### 1. 웹사이트 열기
- `index.html` 파일을 브라우저에서 열기

### 2. 브라우저 호환성
- Chrome, Firefox, Safari, Edge 최신 버전 권장
- IE11 이하는 지원하지 않음

### 3. 로컬 서버 실행 (권장)
```bash
# Python 3가 설치된 경우
python -m http.server 8000

# Node.js가 설치된 경우
npx serve

# 그 후 브라우저에서 http://localhost:8000 접속
```
